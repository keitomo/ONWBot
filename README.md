# Discordワンナイト人狼Bot

## 使い方
1. DiscordのBotアカウントを取得する
2. BotのIntentsの設定をオンにする
3. discord.py 内のTOKENを自分のBotのものに書き換える
4. discordbot.pyを起動する
5. サーバーに招待する
6. サーバーで `/onw start` とコマンドを打つ
7. 遊ぶ
